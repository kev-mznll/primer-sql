# SQL - Práctica y Fundamentos

Repositorio de ejercicios de SQL enfocados en consultas básicas y avanzadas, 
con ejemplos aplicados a un contexto de seguros y actuaría.

## Contenido

- **01_consultas_basicas.sql** — Ejercicios de filtros (`WHERE`), comparaciones, 
  `BETWEEN`, `LIKE`, ordenamiento (`ORDER BY`), agregación (`GROUP BY`, `HAVING`) 
  y `JOIN` entre tablas.

## Contexto de los ejercicios

Las queries simulan una base de datos de aseguradora con tres tablas:

- `asegurados` — información de los titulares de pólizas
- `polizas` — contratos de seguro (vida, auto, etc.)
- `siniestros` — reclamos realizados contra las pólizas

## Temas cubiertos

- Filtros simples y compuestos (`WHERE`, `AND`, `OR`)
- Rangos y patrones (`BETWEEN`, `LIKE`)
- Ordenamiento (`ORDER BY`)
- Funciones de agregación (`COUNT`, `SUM`)
- Agrupación y filtrado post-agregación (`GROUP BY`, `HAVING`)
- Combinación de tablas (`JOIN`)
